Markdown Help

The most common formatting options have keyboard shortcuts to make them easier to add:

Result Markdown
Bold **text**
Emphasize *text*
Strike-through ~~text~~
Link [title](http://)
Inline Code `code`
Image ![alt](http://)
List * item
Blockquote > quote
H1 # Heading
H2 ## Heading
H3 ### Heading
